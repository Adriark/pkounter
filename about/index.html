FAQ / Sobre Pkounter

FAQ / About

Pkounter es un asistente de teambuilding para Champions que combina datos de uso, sets comunes y lectura defensiva/ofensiva para proponer compañeros, counters y ajustes.

Pkounter is a Champions teambuilding assistant that combines usage data, common sets, and offensive/defensive team reads to suggest partners, counters, and set adjustments.

¿De dónde sale la información?

La base competitiva se apoya en MunchStats y datos de Pokémon Showdown/Smogon. Los Pokémon, objetos, habilidades, movimientos y sprites se contrastan con recursos de Pokémon Showdown, PokéAPI y datos locales de Champions.

Where does the information come from?

Competitive data is based on MunchStats plus Pokémon Showdown/Smogon data. Pokémon, items, abilities, moves, and sprites are matched against Pokémon Showdown, PokéAPI, and local Champions data.

¿Cada cuánto se actualizan los datos?

La web refresca sus archivos de datos automáticamente con el script programado. Aun así, las estadísticas públicas del metagame deben tratarse como datos de uso con cadencia realista mensual.

How often is the data updated?

The site refreshes its local data files automatically through the scheduled script. Still, public metagame usage stats should be treated as usage data with a realistic monthly cadence.

¿La app usa IA externa?

Las sugerencias que ves en la web se calculan con lógica local del propio builder: roles, porcentajes de uso, daño estimado, velocidad, tipos, habilidades y movimientos.

Does the app use external AI?

The suggestions shown in the website are produced by local builder logic: roles, usage percentages, estimated damage, Speed, types, abilities, and moves.

He encontrado un bug o una incoherencia

Si ves un set raro, una traducción incorrecta, un dato que no encaja o tienes una sugerencia, escríbeme a contact@pkounter.com.

I found a bug or inconsistency

If you find an odd set, a wrong translation, a data mismatch, or have a suggestion, email me at contact@pkounter.com.

¿Cuál es la web oficial?

La web oficial es pkounter.com. Si alguna ruta o enlace no carga correctamente, prueba desde la portada y avísame en contact@pkounter.com.

What is the official website?

The official website is pkounter.com. If a page or link does not load correctly, start from the homepage and report it at contact@pkounter.com.
