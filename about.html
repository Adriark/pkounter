Pkounter

FAQ / About

Pkounter es un asistente de teambuilding para Champions que combina datos de uso, sets comunes y lectura defensiva/ofensiva para proponer compañeros, counters y ajustes.

Pkounter is a Champions teambuilding assistant that combines usage data, common sets, and offensive/defensive team reads to suggest partners, counters, and set adjustments.

¿De dónde sale la información?

La base competitiva se apoya en MunchStats y datos de Pokémon Showdown/Smogon. Los Pokémon, objetos, habilidades, movimientos y sprites se contrastan con recursos de Pokémon Showdown, PokéAPI y datos locales de Champions.

Where does the information come from?

Competitive data is based on MunchStats plus Pokémon Showdown/Smogon data. Pokémon, items, abilities, moves, and sprites are matched against Pokémon Showdown, PokéAPI, and local Champions data.

¿Cada cuánto se actualizan los datos?

La web puede refrescar sus archivos de datos a diario si el servidor ejecuta el script programado, pero las estadísticas públicas del metagame se consideran mensuales, que es la cadencia realista de actualización de uso.

How often is the data updated?

The site can refresh its local data files daily when the server runs the scheduled script, but public metagame usage stats should be treated as monthly data, which is the realistic usage update cadence.

¿La app usa IA externa?

Las sugerencias que ves en la web se calculan con lógica local del propio builder: roles, porcentajes de uso, daño estimado, velocidad, tipos, habilidades y movimientos. No consume tu ChatGPT al usar la página publicada.

Does the app use external AI?

The suggestions shown in the website are produced by local builder logic: roles, usage percentages, estimated damage, Speed, types, abilities, and moves. Using the published page does not consume your ChatGPT.

He encontrado un bug o una incoherencia

Si ves un set raro, una traducción incorrecta, un dato que no encaja o tienes una sugerencia, escríbeme a contactpkounter@gmail.com.

I found a bug or inconsistency

If you find an odd set, a wrong translation, a data mismatch, or have a suggestion, email me at contactpkounter@gmail.com.
